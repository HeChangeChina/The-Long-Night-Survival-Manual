默认的操作方式如下：\n
W A S D - 移动\n
Shift - 奔跑\n
Space - 翻滚\n
左Shift - 奔跑\n
E - 互动\n
F - 切换互动目标\n

持有武器状态-鼠标右键 - 瞄准\n
瞄准状态-鼠标左键 - 射击\n

数字键1 - 切换至/收起主武器一\n
数字键2 - 切换至/收起主武器二\n
数字键3 - 切换至/收起副武器\n

左Ctrl - 开启技能轮盘\n
技能轮盘-鼠标左键 - 使用技能一\n
技能轮盘-鼠标右键 - 使用技能二\n
技能轮盘-鼠标中键 - 使用技能三\n
技能轮盘-向上滑动鼠标滚轮 - 使用技能四\n
技能轮盘-向下滑动鼠标滚轮 - 使用技能五\n
