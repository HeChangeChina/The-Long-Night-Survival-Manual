默认的操作方式如下：  
W A S D - 移动  
Shift - 奔跑  
Space - 翻滚  
左Shift - 奔跑  
E - 互动  
F - 切换互动目标  

数字键1 - 切换至/收起主武器一  
数字键2 - 切换至/收起主武器二  
数字键3 - 切换至/收起副武器  

持有武器状态-鼠标右键 - 瞄准  
瞄准状态-鼠标左键 - 射击  

左Ctrl - 开启技能轮盘  
技能轮盘-鼠标左键 - 使用技能一  
技能轮盘-鼠标右键 - 使用技能二  
技能轮盘-鼠标中键 - 使用技能三  
技能轮盘-向上滑动鼠标滚轮 - 使用技能四  
技能轮盘-向下滑动鼠标滚轮 - 使用技能五
